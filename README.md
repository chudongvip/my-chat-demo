# Vue3 + Tailwind + RTM = 实时聊天系统

## Setup

### Config your RTM APPID

Press this [link](https://docs.anyrtc.io/cn/AnyRTCCloud/console/manage_projects#%E8%8E%B7%E5%8F%96app-id) to get your RTM APPID.

### Preview

```
cd my-chat-demo
npm run dev
```
